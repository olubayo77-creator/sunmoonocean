Two-phase workflow
Phase 1 — Scrape case list (free-ish)
1. Select a court → click Open Discharge Report Form
2. Log in to PACER if prompted
3. Click Auto-Fill → sets CH13 + Assets + dates
4. Hit Submit on PACER → wait for results table
5. Click Scrape Results Page — gets names + case numbers
6. Repeat for each court

Phase 2 — Enrich for address + dates ($0.10/case)
7. Go to Enrich tab
8. Set batch size (e.g. 25 = ~$2.50)
9. Click Run Enrichment — opens each case docket silently, pulls address + discharge date
10. Repeat batches until all cases enriched
11. Download CSV from Results tab

Tip: You can also load a CSV from a previous session using Load Existing CSV in the Enrich tab.
